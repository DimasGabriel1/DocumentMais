# 📄 Banner Automation & PDF Report

Automação em Python para captura de banners (desktop e mobile) e geração automática de relatórios em PDF.

## 🚀 Sobre o projeto

Este projeto foi desenvolvido para automatizar o processo de prestação de contas de campanhas publicitárias.

A aplicação:
- Acessa sites automaticamente
- Captura banners em versões desktop e mobile
- Evita capturas repetidas (mesmo com banners animados)
- Gera um relatório completo em PDF com:
  - Capa personalizada
  - Comprovante de veiculação
  - Prints capturados
  - Página final com data automática

## 🧠 Tecnologias utilizadas

- Python  
- Playwright  
- PyAutoGUI  
- OpenCV  
- NumPy  
- Pillow  
- ReportLab  
- CustomTkinter  

## 💻 Interface

O projeto conta com uma interface gráfica onde é possível:

- Inserir links (desktop, mobile ou híbrido)
- Definir dados da campanha
- Configurar parâmetros de execução
- Gerar o relatório com um clique
